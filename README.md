# Rock-Paper-Scissors-text-base
hello Guys!

i made a text base game with python:

Rock,Paper,Scissors!

this game has 2 mode : "Player VS Player" and "Player VS PC" 

This game played in several rounds and final winner is the one who wins 3 rounds

and winner of this rounds, winner of game

I will update this game soon and put the graphic version as well

How to Use:
---------------------
1- frist select the game mode : type p for pvp and type e for pvpc, type c to cancel and exit the game

2-if you play pvp, frtist type 1(rock) or 2(paper) or 3(scissors) and then play move for player 2 (1 or 2 or 3) 

3-if you play pve you just select move for 1 player

4-if you type c game is end

5-Keep doing this to reach 3 point (rounds)
